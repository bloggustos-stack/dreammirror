DreamMirror
About this project

Not a corporation. A single mind, with a clear intention.

DreamMirror and Archetype Academy (part of the ONYR.WORLD universe) are built by a single person, with the help of Claude, Anthropic's AI assistant. There is no marketing team, no office, and no hidden commercial agenda behind these tools.

The motivation behind this project is simple: Carl Gustav Jung's analytical psychology — archetypes, the shadow, individuation, the symbolic language of dreams — offers an extraordinarily useful framework for self-knowledge, yet it often remains locked away in academic language or thick books. These tools try to make it accessible, free, and practical, without betraying its depth.

If you have questions, ideas for new archetypes, feature suggestions, or simply want to share what you've discovered about yourself through these tools — the door is open.

Signed by T.R. — the architect of this project.
